Template personnalisé

Vous pouvez si vous le souhaitez personnaliser la structure de la page des tables des matières pour l'adapter à votre thème, ajouter ou supprimer certaines parties, tout cela grâce à l'héritage des thèmes de Dotclear. Il vous suffit simplement d'ajouter dans le répertoire de votre thème le fichier multitoc.html que vous souhaitez voir apparaître. Dotclear se chargera ensuite de prendre ce fichier lors de l'affichage des tables à la place de celui du plugin.

Feuille de style CSS personnalisée

Tout comme pour le template, vous pouvez utiliser une feuille de style CSS personnalisée pour l'affichage des tables des matières en ajoutant le fichier multitoc.css dans le répertoire de votre thème.

Balises de template

Vous avez la possibilité de personnaliser l'affichage de certaines balises de template listées ci-dessous :

  • {{tpl:MultiTocItemDate}}
  • {{tpl:MultiTocItemAuthor}}
  • {{tpl:MultiTocItemCategory}}
  • {{tpl:MultiTocItemNbComments}}
  • {{tpl:MultiTocItemNbTrackbacks}}

Pour chacune de ces balises, vous devez utiliser l'option mask. %s représente le contenu réel. Voici un exemple avec la balise {{tpl:MultiTocItemAuthor}} :
# Retournera le code HTML suivant : <span class="tagada"><a href="lien_auteur">nom_auteur</a></span> {{tpl:MultiTocItemAuthor mask="<span class="tagada">%s</span>" }}
